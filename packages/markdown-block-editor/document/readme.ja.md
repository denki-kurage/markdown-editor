# Markdown Blcok Editor の使い方

## このブロックエディタの基本

![](./md-00.png)

ブロックエディタのリストから`Markdown with Block Editor`を探して追加します。


![](./md-01.png)

左のVSCode風のエディタでマークダウンを記入します。
右にプレビューが表示されます。

![](./md-02.png)

文字列を選択して「B」アイコンをクリックすると太字にすることができます。

![](./md-03.png)

保存して、実際に編集されたものをフロントエンドのページを確認してください。

![](./md-04.png)

次は画像を追加します。


![](./md-05.png)

プレビューで投稿した画像が確認できます。



## ちょっと高度な機能

テーブル作成やソースコード記入も簡単にできます。

![](./md-06.png)

例えばテーブルを追加したい場所に`3x`と入力するとインテリセンスが働きます。
「横３列、縦４行」のテーブルを追加したい場合`3x4`を選びます。

![](./md-07.png)

テーブルのひな型が作成されます。

さっきのカレー具材リストをテーブルにしてみましょう。

* そのままでは１行足りないので追加してください。


![](./md-08.png)


テーブルを入力していきます。
そのままテーブルを編集すると見づらいですが、テーブルを成形することができます。


![](./md-09.png)

成形ボタンを表示するにはテーブルの文字列ではない場所(例えば縦線の`|`)を選択してください。
ボタンをクリックするとテーブルは成形され、きれいに見えます。

ただし、選択されたフォント次第ではレイアウトが崩れます。
これは文字列が等幅フォントになってないからです。

OSなどの環境、言語によって有効な等幅は変わってきます。
適切なフォントを左の設定項目から調整してください。



![](./md-10.png)

`#`を入力するとヘディングを追加するためのインテリセンスが使用できます。


![](./md-11.png)

(`)や(~)を入力するとソースコード追加支援が使用できます。


![](./md-12.png)

ソースコードのハイライトにはPrism.jsが使用されてます。
